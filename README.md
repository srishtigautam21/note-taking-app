# notet - A note-taking-app

## Features

### P0

- Landing Page
- Notes Main Page
- Filter by:
  - Date
  - Tags
  - Priority
- Trash Management
- Archive Management
- Edit Notes
- Add colors to notes
- Basic authentication:
  - Login
  - Logout

## Tech Stack

**Client:** React, Umbrella-UI CSS

**Server:** Mockbee
